# Higher or Lower - Python Game

Guess which personality has more followers and see how high you can score.  
**Note:** All data in this project is completely made up for fun and learning purposes.

## Features
- Fun, text-based gameplay in the terminal
- Keeps track of your score
- Clean ASCII art for visuals
- Continuously challenges you with new options

## How to Play
1. Run the Python script: `python higher_lower.py`
2. Choose between option 1 or 2 by typing `1` or `2`
3. Correct guesses add to your score
4. Game ends when you pick the wrong option
5. Try to beat your high score!

## Requirements
- Python 3.x
- `art.py` module included
- `game_data.py` containing fictional follower data
